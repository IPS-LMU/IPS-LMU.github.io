EMU Speechdatabase Management System

Warning! The software that is described below is not yet available to the public. The following text is meant as a overview of the future system architecture. If parts of the project are released it will be announced here. So please check back here in the near future. We hope to release parts of the project as soon as possible

We are currently working on the next iteration of the EMU speechdatabase management system. It will be mainly based in the R language for statistical computing. Two R packages (the gray items in the Figure below) are currently being developed to complete the system, one being the wrassp package to handle the speech signal processing and the other handling database manipulation and querying as well as serving as a local websocket server to connect to the webApp/GUI. This infrastructure will allow researchers to have a self-contained system to query databases, perform signal processing and perform statistical evaluations all within the same environment. Being autonomous, however wrassp and the web application will also be able to be used separately if desired.

Installation instructions:

There are several different ways to get the EMU-webApp running on your system. However, for the time being, we will only support one of these methods: Download and install Google-Chrome (support for other browsers will follow!). Next point chrome to the following URL: http://raphywink.github.io/emuLVC/. Thats it! This will download and cache the EMU-webApp for online and offline usage. If you ever have to use the EMU-webApp in an offline situation just navigate to the above URL and the browser will load the application from the application cache.

Warning! Packages below are not yet available on CRAN!!

To install both the EMU-rPackage as well as the wrassp-rPackage only the EMU-rPackage has to be installed. This is achieved by typing install.packages("EMU-rPackage")into the R command prompt.
