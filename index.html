The EMU speechdatabase management system will be based in the R language for statistical computing. Two R packages (the gray items in the Figure below) are currently being developed to complete the system, one being the wrassp package to handle the speech signal processing and the other handling database manipulation and querying as well as serving as a local websocket server to connect to the webApp/GUI. This infrastructure will allow researchers to have a self-contained system to query databases, perform signal processing and perform statistical evaluations all within the same environment. Being autonomous, however wrassp and the web application will also be able to be used separately if desired. The tools / projects presented here are still in early development and are have not made public yet. We are working hard to make these project public and open as soon as possible.
